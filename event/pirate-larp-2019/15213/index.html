Pirate LARP 2019

📅 19. 07. 2019 — 21. 07. 2019

📍 Středočeský, Poděbrady

👥 50 hráčů

Web: http://piratskylarp.4fan.cz/

Popis Pirate Larp 2019 Vítám vás příznivci všech pirátů a všechny vás kdo máte rádi 17, 18 století. Tento ročník bude jiný než předešlé 2. Hodně jsme si dali záležet na tom aby se vám to líbilo a užili jsme si srandu. maximální počet hráčů je 50 z důvodů předpisů, velikosti místa pro spaní a domluvy s městem. Počet účastníků je možné navýšit ale pouze ve vyjímečném případě, po domluvě s organizatory.   datum akce: 19 - 21. července 2019   Stránky akce: www.piratelarp.cz Facebook: https://www.facebook.com/piratelarp/ e-mail: piratelarp [at] gmail [dot] cz
